# 🎂 Birthday Message Generator
An interactive web app that generates fun and personalized birthday messages based on user input. Built using HTML, CSS, Bootstrap, and JavaScript with engaging UI effects.
## 🚀 Features

* 🎉 Generate birthday messages (Nice / Funny)
* ⏳ Countdown to birthday (days remaining)
* 🎊 Confetti animation for celebration
* 🔊 Sound effects for better experience
* 🎨 Modern glassmorphism UI with Bootstrap
* ⚡ Smooth animations and transitions
## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

## 🧠 How It Works

1. User selects a birthday date
2. Chooses message type (Nice / Funny)
3. Clicks **Generate**
4. App:

   * Calculates days remaining
   * Displays a message
   * Shows confetti animation
